# Chrome Web Store Listing — X 清道夫

## 短描述 (≤132 字符)

在 X 关注列表页筛选未回关用户，一键勾选并批量取消关注。

---

## 详细描述

**X 清道夫** 是一款轻量级 Chrome 扩展，帮助你高效清理 X/Twitter 的关注列表。

### ✨ 核心功能

- **智能筛选** — 一键切换「只看未回关」，快速定位不互相关注的账号
- **一键全选** — 自动勾选当前页面已加载的所有用户
- **批量取关** — 确认后自动依次取消关注，带实时进度条
- **纯本地运行** — 所有操作在你的浏览器中完成，不上传任何数据，不收集任何信息

### 🚀 使用步骤

1. 打开 X/Twitter 上**你自己的** Following 页面（`x.com/你的用户名/following`）
2. 扩展自动激活，页面顶部出现浮动工具栏
3. 使用筛选器找到未回关的用户
4. 勾选你要取关的用户（或点击「全选」）
5. 点击「批量取关」，确认弹窗后等待完成

### ⚠️ 注意

- 仅在 `x.com/*/following` 和 `twitter.com/*/following` 页面生效
- 不会在他人关注列表上运行
- 每次取关间隔 1 秒，避免触发 X 速率限制
- 浏览器本地操作，零数据收集
